# Luka Loehr - Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript. The site features a modern, responsive design with a dark theme, vibrant gradients, and smooth animations.

## Features

- Fully responsive design that works on all devices
- Dark theme with vibrant gradient backgrounds
- Modern typography using Inter font family
- Scroll-based animations and hover effects
- Interactive navigation with smooth scrolling
- Mobile-friendly with hamburger menu
- Contact form with validation
- Animated hero section background

## Tech Stack

- HTML5
- CSS3 (with CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (no frameworks)
- Font Awesome for icons

## Sections

1. **Hero Section** - Introduction with animated gradient background
2. **About** - Personal bio and skills
3. **Tech Stack** - Grid of technologies with hover effects
4. **Projects** - Showcase of key projects
5. **Contact** - Form to get in touch

## Customization

The site can be easily customized by modifying:

- Colors and gradient values in CSS variables (`:root` section in style.css)
- Content in index.html
- Animations and interactions in script.js

## How to Use

1. Clone the repository
2. Open index.html in your browser
3. Modify the content as needed
